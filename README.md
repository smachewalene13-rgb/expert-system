# 🏥 AI Medical Diagnosis System

An intelligent medical diagnosis assistant that uses AI to analyze symptoms and provide preliminary diagnoses.

## Features

- 🤖 **AI-Powered Diagnosis** - Uses TF-IDF and symptom matching algorithms
- 👤 **User Authentication** - Patient, Doctor, and Admin roles
- 📊 **Dashboard** - Personal health dashboard with history
- 📝 **PDF Reports** - Generate professional medical reports
- 📧 **Email Notifications** - Send diagnosis reports via email
- 📈 **Analytics** - Visualize symptom patterns and trends
- 🔒 **Secure** - Password hashing and session management

## Tech Stack

- **Backend**: Flask (Python)
- **Database**: SQLite / PostgreSQL
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **AI**: scikit-learn, NumPy, NLTK
- **Reporting**: ReportLab
